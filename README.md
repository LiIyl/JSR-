## 削除・利用できなくなった内容

### バニラ車両の販売を終了

現在バニラ車両を所持している方は、引き続きそのままお使いいただけます。

今回のアップデート以降、新たにバニラ車両を購入することはできなくなりました。

ただし、MOD車両との兼ね合いにより、街中を走行しているNPC車両や一部のジョブでは、引き続きバニラ車両が使用されます。そのため、バニラ車両に乗ること自体は可能です。
